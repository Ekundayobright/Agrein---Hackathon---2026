# 🌱 Agrein

## 🚀 Overview
Agrein is a social-commerce platform designed to connect farmers directly with buyers. It allows farmers to share farm updates, showcase products, and sell securely, while buyers can discover agricultural products, interact with posts, and make purchases within the platform.

The goal is to improve transparency, reduce middlemen, and create a trusted digital marketplace for agriculture.

---

## 🎯 Key Features

### 🌍 Social Feed System
- Farmers can create posts (images, videos, farm updates)
- Users can like and comment on posts
- Engagement-driven agricultural community

### 🛒 Product Catalog
- Farmers can upload products with:
  - Price
  - Quantity
  - Description
  - Images
- Real-time inventory tracking
- Stock validation before checkout

### 💳 Secure Payments
- Integrated with Interswitch API
- Secure transaction processing
- Payment confirmation and receipts
- Optional escrow system for trust

### 📍 Location & Delivery
- Location capture for farmers and buyers
- Delivery address system
- Future integration with logistics providers

### 🛡️ Security & Trust
- Farmer verification system
- Secure authentication (JWT/OAuth)
- No direct messaging (reduces fraud)
- Transaction history tracking
- Rating and review system

### 🧑‍💼 Admin Dashboard
- User verification
- Post moderation
- Fraud monitoring
- Analytics and reporting

---

## 👥 User Roles

### 👨‍🌾 Farmer

### 🛍️ Buyer
